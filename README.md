# lumen-snark

[![CI](https://github.com/Tranduy1dol/lumen-snark/actions/workflows/ci.yml/badge.svg)](https://github.com/Tranduy1dol/lumen-snark/actions/workflows/ci.yml)

> ✨ Part of the [luminescent](https://github.com/Tranduy1dol/luminescent) project — *Illuminating the path to zero-knowledge*

Study notes and implementations from [Proofs, Arguments, and Zero-Knowledge](https://people.cs.georgetown.edu/jthaler/ProofsArgsAndZK.html) by Justin Thaler.

> ⚠️ **Educational Purpose Only**: This library is for learning purposes and is **NOT production-ready**.

## Features (Planned)

- Interactive proofs (IPs)
- Multi-prover interactive proofs (MIPs)
- SNARKs and their variants
- Commitment schemes

## Related Projects

This library is part of the [luminescent](https://github.com/Tranduy1dol/luminescent) learning monorepo for Zero-Knowledge Proofs.
